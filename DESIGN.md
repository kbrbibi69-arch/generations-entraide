# Design System: Générations Entraide
**Project ID:** 9311637904765247837

## 1. Visual Theme & Atmosphere
The design system of Générations Entraide is **warm, professional, and accessible**. It uses a "Modern Solidarité" aesthetic, balancing clean white surfaces with warm earth-tone accents and glassmorphism elements to create a sense of transparency and trust.

## 2. Color Palette & Roles
*   **Primary Beige Nude (#EADDD8)**: Used for soft backgrounds and subtle highlights.
*   **Secondary Marron Terre (#5D4037)**: Primary brand color for headings, footers, and major UI text.
*   **Global Cream Background (#FDFBF7)**: Base surface color for all pages.
*   **Vibrant Orange Accent (#F27F0D)**: Used for primary calls to action, icons, and focus elements.
*   **Deep Anthracite (#333333)**: Secondary text and high-contrast elements.

## 3. Typography Rules
*   **Font Family**: `Outfit` (Headings) and `Inter` (Body).
*   **Headings**: Bold, tracking-tight, marron terre color.
*   **Body**: Medium weight, anthracite or gray-600, high legibility.

## 4. Component Stylings
*   **Buttons**: 
    *   Primary: Gradient Orange-to-Yellow (#f27f0d to #eab308), full-pill shape, subtle shadow.
    *   Secondary: Glassmorphism white/20 with border, backdrop blur.
*   **Cards**: 
    *   Glassmorphism: White/60 background, backdrop-blur 20px, 40px rounded corners (3xl/4xl), subtle border-white/40.
*   **Icons**: Lucide icons in orange-500 or white.

## 5. Layout Principles
*   **Whitespace**: Generous vertical padding (py-24) to ensure focus on content.
*   **Alignment**: Centered hero sections, 3-column grids for features and actions.
*   **Transitions**: Smooth Framer Motion entries (y: 0, opacity: 1) and parallax scaling on backgrounds.
